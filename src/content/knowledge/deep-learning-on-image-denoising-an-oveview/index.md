# 基础的CNN发展过程
- LeNet：证明卷积神经网络可以不依靠特征工程
- AlexNet：更深的网络，使用了更大的卷积核，受限于GPU，当时还得两个GPU并行运算
- VGG：以块的概念设计，VGG块为：卷积+非线性激活+汇聚
- NiN：以$1\times 1$卷积层出名，不改变空间结构实现了全连接
- GoogLeNet：以Inception块中，多个大小的卷积核得到的特征合并出名
- BN：批量标准化，人为干预特征，减小了模型对少量特殊的样本的敏感性
- ResNet：残差学习出名，缓解了深度学习中的梯度消失问题

### CNN/NN
#### 问题与方法
自动特征提取方法在图像应用中对减小计算量有重要作用。

- DnCNN：处理多样的低质量图像任务，如图像降噪，超高清分辨率，去除方块滤波，使用了以下结构：
  - 基于CNN
  - 批量标准化（BN）：一种处理输出的方式，一般夹在卷积之后，非线性激活函数之前，尚未从理论证明其效果，但实践表明，该方法可以加快训练速度，降低过拟合程度；该方法存在较大争议。
  - 残差学习技术（ResNet提出）：简单来讲，其效果是避免梯度消失，使得深层次的神经网络能够以更快地速度下降损失函数（靠近输入的块的参数能够以更快的速度更新）；具体而言，该技术使得样本能够以更快的速度向前传播

除此外，还有许多利用残差学习技术来使得深度卷积神经网络能够处理图像降噪问题

但是过深的CNN会出现长期依赖问题（没懂，这个一般是RNN考虑的事情）
- 利用recursive and gate units，门控循环单元，没了解过
- 提出一种用以解释信号处理与深度学习关系的卷积框架，不懂

为了解决信息不充足的图像（比如医疗图像）相关问题
- 深度卷积神经网络、残差学习技术与多尺度技术
  - 其中多尺度技术，一般有两种
    - 多尺度输入，如多分辨率输入
    - 多尺度特征，如GoogLeNet中Inception块，不同卷积核抓取不同的特征，并组合在一起

然而，提出的解决这类问题的模型出现了计算量增大与内存消耗增大的问题。

- 有人使用空洞卷积处理该问题，增大了感受野并降低了深度
  - 空洞卷积计算网上有很多图解
  - 感受野：经过卷积得到的特征图上的一点相较于原图的像素，计算为：$F_i = (F_{i-1}-1)\times stride +k$，$F_i$代表第几层相对于原图的感受野，$k$是卷积核大小
  - 空洞卷积扩大了感受野，空洞的卷积核$k' = k+(k-1)(r-1)$，r为膨胀因子，是空洞程度的描述，更新感受野公式为：$F_i = (F_{i-1}-1)\times stride +k'=(F_{i-1}-1)\times stride+k+(k-1)(r-1)$
  - 空洞卷积会丢失信息
- 有人提出一种残差神经网络，通过多尺度交叉拼接，处理医学图像，没看过
  - Multi-scale Cross-path Concatenation Residual Network for Poisson denoising

上述大部分方法都是基于改进CNN来解决问题。

#### 改进方法
- 从多样的输入融合特征
  - 将同一样本的不同部分作为多样的输入
  - 将不同角度的同一样本作为输入
  - 将CNN的不同通道作为输入
- 改变损失函数
  - jointed Euclidean and perceptual loss functions获取更多边缘信息，没详细了解过
- 增加CNN的深度（加模块）或宽度（模块里并行卷积，最后输出拼接）
- 引入新的模块
  - 激活函数、全连接层、汇聚层（池化层）等
- 引入skip connection和级联操作
  - skip connection：解决梯度消失问题的
    - 相加：比如ResNet将输入提前送到模块结束处，相加；形状不同时使用$1\times 1$卷积层
    - 还有些方法采用拼接，没详细看过，因为拼接容易让参数增多，计算量增大，而又没有多尺度的效果好
  - 没看过级联操作的内容
### CNN/NN和普通的特征提取方法
**这一部分内容，我之前从没接触过**

特征工程对于传统机器学习非常重要。

但是深度学习时黑盒子技术（你不知道它提取了什么玩意），不允许人为地选择特征。

将特征工程一定形式塞入卷积神经网络？

- 对于weak edge-information noisy images来说，CNN with transformation domain method来移除噪声是非常有效的。
- 对于non-linear noisy images来说，CNN with kernel method在恢复潜在干净图像是非常有效的。
  - CNN提取特征
  - 核方法将非线性特征转成线性
  - 配合残差学习，重构潜在的干净图像
- 对于high dimensional noisy images来说，CNN和降维方法的组合是常用的去噪方法。
- 对于non-salient noisy images来说，信号处理方法能引导CNN来提取显著的特征。
- 对于high computational cost， CNN和图像的属性结合能有效地降低复杂度。

### optimization和CNN结合
机器学习使用优化方法和判别学习来处理图像问题
- 优化方法对于不同的低质量的图像任务有不错的效果，但是为了优化，需要人为设置参数，放在深度卷积里要算死人，花时间
- 判别学习不适合低质量图像问题，不够灵活

为了在效率与灵活度直接达到一个微妙的平衡，提出：
- 基于优化方法的判别学习
- 有先验知识的CNN，比如在损失函数里添加正则项，对参数进行人为的约束

    - 对参数引入拉普拉斯先验等于加了$L_1$正则化
- 提高降噪速度，优化+CNN
